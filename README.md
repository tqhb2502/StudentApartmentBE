## 2. Cài đặt dự án

### 2.1. Cài đặt package
    composer install

### 2.2. Tạo key dự án
    php artisan key:generate

### 2.3. Tạo database và kết nối
> Tạo database với tên student_apartment_db
> Duplicate tệp .env.example, đổi tên thành .env

### 2.4. Migrate database
    php artisan migrate

### 2.5. Chạy các file seeder tạo dữ liệu mẫu
    php artisan db:seed

### 2.6. Chạy dự án
    php artisan serve
